# 仏教変容プロトコルの四段階構造マップ

## 原始仏教 → 上座部仏教 → 大乗仏教 → 密教

認識OSのアップデート過程として仏教を捉え、各段階の「上位概念 → 中位理論 → 下位実践（具体的行動）」を一覧可能な形で構造化する。

-----

## 0. 全体マップ：四段階のOS進化

|段階       |OSの位置づけ              |中核問題           |解決アプローチ       |到達点    |
|---------|---------------------|---------------|--------------|-------|
|**原始仏教** |純粋なソースコード（ブッダ直説）     |生存に付随する「苦」の因果解明|四諦による分析 → 八正道 |苦の滅尽・涅槃|
|**上座部仏教**|長老による解釈固定化OS（クローズド仕様）|実装の再現性確保       |三学・七清浄による段階的精錬|阿羅漢位   |
|**大乗仏教** |普遍インターフェースへの拡張       |個人解脱の閉鎖性の打破    |空・唯識による主客の同時解体|菩薩道・転依 |
|**密教**   |身体性を統合した最新OS         |理論と実践の時間的分断    |三密加持による即時同期   |即身成仏   |

各段階は前段階を否定するのではなく、**基盤として継承しつつ対象範囲と実装精度を拡張する**関係にある。

-----

## 1. 原始仏教：苦の因果プロトコル

### 1-1. 階層構造

```
【上位概念】四諦（したい）= 苦の結果と原因、楽の結果と原因の因果律
    │
    ├─【中位理論】
    │   ├─ 苦諦：現状分析（四苦八苦 / 五蘊盛苦）
    │   ├─ 集諦:原因特定（渇愛トリシュナー / 行サンカーラ）
    │   ├─ 滅諦:理想状態（涅槃 / 中道）
    │   └─ 道諦:実践体系(八正道)
    │
    └─【下位実践】八正道(行動指針)
```

### 1-2. 八正道：具体的行動チェックリスト

|項目 |領域|具体的行動           |
|---|--|----------------|
|正見 |認知|四諦の因果律で事象を捉える   |
|正思 |思考|執着・怒り・害意を排した思考  |
|正語 |言語|真実・有益な言葉のみ伝達    |
|正業 |行動|生命・財産・倫理を損なわない行為|
|正命 |生計|道徳的・合理的な生計手段    |
|正精進|努力|善の増進、悪の抑止       |
|正念 |観察|心身状態の常時モニタリング   |
|正定 |集中|深い集中による認知解像度の向上 |

**核心メッセージ**：自己中心（天動説）から法中心（地動説）へのコペルニクス的転回。

## 2. 上座部仏教：段階的精錬プロトコル

### 2-1. 階層構造

```
【上位概念】三学(戒・定・慧) = 解脱への到達度KPI
    │
    ├─【中位理論】七清浄(段階的進捗管理)
    │   ├─ 止の段階:戒清浄 → 心清浄
    │   └─ 観の段階:見清浄 → 度疑清浄 → 道非道智見清浄
    │                     → 行道智見清浄 → 智見清浄
    │
    ├─【認識深化】三遍知(知遍知 → 度遍知 → 断遍知)
    │
    └─【下位実践】観智9段階のシーケンス
```

|3|怖畏智  |現象を拠り所なき恐ろしいものとして認識|
|4|過患随観智|現象の過失と危険性を見出す      |

|6|脱欲智  |束縛からの解放を切望         |

## 3. 大乗仏教：主客同時解体プロトコル

### 3-1. 階層構造

```
【上位概念】空(śūnyatā) = 人法二無我 / 主客同時解体
    │
    ├─【中位理論A】中観:カマラシーラ4段階認識変容
    │   煖位・頂位 → 忍位 → 世第一法 → 初地
    │
    ├─【中位理論B】唯識:八識構造と三性説
    │   遍計所執性 → 依他起性(似有) → 円成実性
    │   薫習メカニズム → 転依(āśraya-parāvṛtti)
    │
    ├─【中位理論C】華厳:事事無礙(有力・無力の力動)
    │
    └─【下位実践】六波羅蜜(菩薩道の行動指針)
```

### 3-2. カマラシーラ認識変容の4段階

|修行位   |三昧     |認知操作              |
|------|-------|------------------|
|煖位・頂位 |明得・明増三昧|外界対象が識と別個に存在しないと考察|
|忍位    |一分入三昧  |客体の無顕現、唯心への収束     |
|世第一法  |無間三昧   |主客二分を離れた不二智の顕現    |
|初地(見道)|—      |不二智さえも無自性として超越    |

### 3-3. 六波羅蜜：菩薩道の行動チェックリスト

|項目|機能    |具体的行動            |
|--|------|-----------------|
|布施|リソース循環|余剰を手放すことで「豊かさ」を確定|
|持戒|原則確立  |アイデンティティをノイズから防護 |
|忍辱|耐性    |逆境でシステムを停止させない   |
|精進|駆動    |最短経路で歩みを止めない     |
|禅定|集中    |リソースを一点に集約       |
|智慧|メタ視座  |前5項の統合から本質を見抜く   |

**核心メッセージ**：依他起性を「似有」として戦略的に措定することで、修行の連続性と転依を担保する。

## 4. 密教：身体統合即時プロトコル

### 4-1. 階層構造

```
【上位概念】即身成仏 = 身体を介した大日如来との同期
    │
    ├─【中位理論A】三密加持
    │   身密(印) / 口密(真言) / 意密(観念)
    │
    ├─【中位理論B】両部曼荼羅
    │   胎蔵界(理・慈悲) / 金剛界(智・神通神変)
    │
    └─【下位実践】五相成身観 5段階
```

### 4-2. 五相成身観：具体的5ステップ

|#|段階   |真言                                 |実装                 |
|-|-----|-----------------------------------|-------------------|
|1|通達菩提心|Om cittaprativedham karomi         |阿頼耶識奥底の月輪を「朧月」として観る|
|2|修菩提心 |Om bodhicittam utpadayami          |月輪を16分割で磨き「満月」に完成  |
|3|成金剛心 |Om tiṣṭha vajra                    |月輪中に五鈷金剛杵(太陽)を顕現   |
|4|証金剛身 |Om vajrātmako ’ham                 |金剛杵が全身に浸透、入我       |
|5|仏身円満 |Om yathā sarvatathāgatās tathā ’ham|大日如来との入我我入の完成      |

### 4-3. 三密の実装チェックリスト

- **身密**：印(Mudra)の結節点、脊髄の垂直性を保持
- **口密**：真言の振動を全身に響かせ宇宙周波数と同期
- **意密**：阿字観で仮我を解体、月輪→金剛杵のプロセスを観想

**核心メッセージ**：「三劫成仏」という漸進モデルを破壊し、宇宙の体・相・用・無礙の一体性を「今ここ」で再発見する。

## 6. 実装プロセスとしての連続性

四段階は独立した選択肢ではなく、**介入する認識レイヤーの深化**として連続している。

```
原始仏教 ─ 第6意識(概念)レベルで因果を理解
    ↓
上座部  ─ 同レベルを段階的に精錬し深層へ接近
    ↓
大乗    ─ 第7末那識・第8阿頼耶識を書き換える(転依)
    ↓
密教    ─ 第9阿摩羅識・身体を統合し宇宙と同期(即身成仏)
```

言語(第6意識)でいくら理論を理解しても、末那識・阿頼耶識はデバッグできない。これが密教が身体的実践(フィジカル・デバッグ)を要請する論理的必然性である。

-----

## 7. 結語

四段階はそれぞれ、**異なる深度の認識レイヤーに介入する専用プロトコル**である。現代の実装者は、自らの問題がどの深度に存在するかを見極め、対応する段階のツールを戦略的に選択・統合することが求められる。

- 表層の行動ノイズ → **原始仏教(八正道)**
- 段階的な精神精錬 → **上座部(観智)**
- 深層の認知バイアス → **大乗(六波羅蜜・転依)**
- 身体まで含めた全方位変容 → **密教(五相成身観)**

世界は認識次第で書き換え可能である。問うべきは「どのレイヤーから介入するか」の一点に尽きる。
