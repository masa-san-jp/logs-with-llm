# Google Chat Logger セットアップガイド

このガイドでは、Google Apps Scriptを使用して、送信したGoogle Chatのメッセージを自動的にGoogleスプレッドシートに記録する方法を説明します。

## 前提条件
- Google Workspace アカウント（推奨）。
- Google Cloud Platform (GCP) プロジェクトへのアクセス権（Google Chat APIを使用するために必要）。

## セットアップ手順

### 1. Google スプレッドシートの作成
1. 新しいGoogle スプレッドシートを作成します。
2. 名前を「Chat Log」（または任意の名前に）変更します。
3. **拡張機能** > **Apps Script** を開きます。

### 2. コードの追加
1. デフォルトの `Code.gs` の内容を削除します。
2. このフォルダにある [Code.gs](./Code.gs) の内容をコピーし、スクリプトエディタに貼り付けます。
3. `Setup.gs` という名前で新しいスクリプトファイルを作成します。
4. [Setup.gs](./Setup.gs) の内容をコピーし、そこに貼り付けます。
5. プロジェクトを保存します (Ctrl+S)。

### 3. Google Cloud プロジェクトのリンク (重要)
Google Chat APIを使用するには、Apps Scriptで作成されるデフォルトのプロジェクトではなく、標準のGCPプロジェクトが必要です。
1. [Google Cloud Console](https://console.cloud.google.com/) にアクセスします。
2. 新しいプロジェクトを作成します（例: "Chat Logger"）。
3. ダッシュボードから **プロジェクト番号** (IDではありません) をコピーします。
4. Apps Scriptエディタに戻り、**プロジェクトの設定** (歯車アイコン) を開きます。
5. 「Google Cloud Platform (GCP) プロジェクト」の下にある **プロジェクトを変更** をクリックします。
6. プロジェクト番号を貼り付けて、**プロジェクトを設定** をクリックします。

### 4. Google Chat API の有効化
1. 作成したプロジェクトの Google Cloud Console で、**API とサービス** > **ライブラリ** に移動します。
2. **Google Chat API** を検索して有効にします。
3. **API とサービス** > **OAuth 同意画面** に移動します。
   - **内部** (可能な場合) または **外部** を選択します。
   - 必須項目（アプリ名、ユーザーサポートメール、開発者の連絡先情報）を入力します。
   - 保存して次へ進みます。

### 5. Apps Script での拡張サービスの追加
1. Apps Scriptエディタの左サイドバーにある **サービス** の横の **+** アイコンをクリックします。
2. **Google Chat API** を選択します。
3. **追加** をクリックします。

### 6. 初期化
1. ツールバーのドロップダウンメニューから `setup` 関数を選択します。
2. **実行** をクリックします。
3. 権限の承認を求められたら、許可します。

### 7. 実行テスト
1. Google Chat で新しいメッセージを送信します。
2. Apps Script で `logMyMessages` 関数を選択します。
3. **実行** をクリックします。
4. Google スプレッドシートを確認し、メッセージが表示されているか確認します。

### 8. 自動化 (トリガー設定)
1. Apps Script で **トリガー** (目覚まし時計のアイコン) に移動します。
2. **+ トリガーを追加** をクリックします。
3. 実行する関数: `logMyMessages`。
4. イベントのソース: **時間主導型**。
5. 時間ベースのトリガーのタイプ: **分ベースのタイマー**。
6. 時間の間隔: **10 分おき** (またはお好みの間隔)。
7. 保存します。

これで、メッセージが自動的に記録されるようになります！
