# 補助金・助成金獲得パイプライン

## 全体フロー概観

```
[1. 情報収集・調査] → [2. 適格性評価] → [3. 申請準備] → [4. 申請・審査対応] → [5. 採択後管理] → [6. 完了・次サイクル]
```

---

## Phase 1｜情報収集・調査

### 1-1. ソース登録・モニタリング
- 公的機関（経産省・中小機構・都道府県・市区町村）の公募情報を定期チェック
- J-Net21、ミラサポplus、e-Gov などのポータルをRSS・メールで購読
- 業界団体・商工会議所からの案内を受信

### 1-2. 公募情報の一次整理
| 収集項目 | 内容 |
|---|---|
| 補助金名 | 正式名称・通称 |
| 主管機関 | 省庁・自治体・財団 |
| 対象者要件 | 業種・規模・設立年数 など |
| 対象経費 | 設備・人件費・委託費 など |
| 補助率・上限額 | 例：2/3補助・上限1,000万円 |
| 公募期間 | 締切日・受付回数 |
| 採択率（過去実績） | あれば取得 |

---

## Phase 2｜適格性評価（スクリーニング）

### 2-1. 申請資格の確認
- 法人格・業種コード（JSIC）の該当確認
- 従業員数・資本金による中小企業判定
- 設立年数・業歴要件
- 過去の受給歴・重複申請制限の確認

### 2-2. 事業適合度の評価
- 自社の計画・課題と補助対象事業の一致度をスコアリング
- 採点基準（審査項目）と自社の強みの対応付け
- 費用対効果試算（申請工数 vs 獲得見込み額）

### 2-3. 優先順位付け
- スコア上位案件を申請候補としてリスト化
- 締切・準備期間を考慮したタイムライン確認

---

## Phase 3｜申請準備

### 3-1. 事業計画の策定・ブラッシュアップ
- 現状課題の定量的整理（売上・生産性・コスト など）
- 補助事業の目的・実施内容・期待効果の記述
- KPI・数値目標の設定（採択審査で重視される）
- 競合比較・市場環境の記述（必要に応じて）

### 3-2. 必要書類の収集・作成
| カテゴリ | 代表的な書類 |
|---|---|
| 法人基本 | 登記簿謄本・定款・印鑑証明 |
| 財務 | 直近2〜3期の決算書・納税証明 |
| 事業計画 | 申請書本体・事業計画書・実施スケジュール |
| 見積 | 対象経費の相見積書（2社以上が多い） |
| その他 | 宣誓書・同意書・特定書類（補助金ごとに異なる） |

### 3-3. 予算・資金計画の整備
- 補助対象経費と対象外経費の分離
- 補助金受給前の立替払い計画（多くは後払い）
- 金融機関との事前相談（必要に応じて融資と組み合わせ）

### 3-4. 申請書レビュー・品質確認
- 審査基準に対する自己採点
- 第三者（社内・外部専門家）によるレビュー
- 提出前チェックリストの消込

---

## Phase 4｜申請・審査対応

### 4-1. 申請提出
- 電子申請（jGrants 等）または郵送・窓口提出
- 提出完了の証跡保存（受付番号・受領確認メール）
- 提出期限の厳守（時間指定がある場合は注意）

### 4-2. 審査期間中の対応
- 補足資料・追加説明の要求に迅速対応
- 審査員ヒアリング・プレゼンへの備え（審査形式による）
- 採否見込みに応じた並行申請の検討

### 4-3. 採否通知の確認
- 採択通知：交付申請手続きへ移行
- 不採択通知：審査講評の入手・改善点の記録

---

## Phase 5｜採択後管理（交付〜実績報告）

### 5-1. 交付申請・交付決定
- 交付申請書の提出（採択≠交付決定、必ず手続きが必要）
- **交付決定前の発注・契約・支払いは原則補助対象外**（最重要）
- 交付決定通知の受領と内容確認

### 5-2. 事業実施・経費管理
- 補助事業計画に沿った実施と進捗管理
- 経費の証憑管理（請求書・領収書・振込記録の一式保管）
- 変更が生じた場合は事前に変更承認申請

### 5-3. 実績報告・精算
- 完了報告書・実績報告書の提出
- 経費精算・補助金額の確定
- 確定通知後に補助金の入金

### 5-4. 事後モニタリング対応
- 補助事業完了後の報告義務（多くは3〜5年間継続報告）
- 設備の転用・処分制限の遵守
- 調査・立入検査への対応

---

## Phase 6｜完了・ナレッジ蓄積・次サイクル

### 6-1. 振り返り・改善
- 採択・不採択要因の分析
- 申請書・事業計画書の改善版の保存
- 申請工数・コストの記録

### 6-2. ナレッジ管理
- 申請書テンプレートの整備
- 補助金ごとの審査ポイントの社内ドキュメント化
- 専門家（中小企業診断士・行政書士）との関係構築

### 6-3. 次サイクルへの接続
- 翌年度公募の先行調査
- 事業計画の更新と新規補助金との適合性確認
- 申請候補リストの更新

---

## 主要な落とし穴（リスク一覧）

| リスク | 内容 | 対策 |
|---|---|---|
| 交付決定前発注 | 補助対象外となり全額自己負担 | 交付決定日を必ず確認してから発注 |
| 証憑不備 | 精算時に経費が認められない | 発生都度、証憑セットで保管 |
| 報告義務漏れ | 返還請求・指定取消のリスク | 報告スケジュールをカレンダー管理 |
| 重複受給 | 同一経費への複数補助金の適用不可 | 採択済み補助金との経費区分を明確化 |
| 申請書の抽象的記述 | 審査で低評価 | 数値・具体的根拠で記述 |
