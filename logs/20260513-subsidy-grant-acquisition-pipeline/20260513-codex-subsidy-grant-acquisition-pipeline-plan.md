# 補助金・助成金獲得パイプライン構築計画

## 0. 前提

本計画は、以下の前提で設計する。

```text
目的：
自組織・自事業の情報を構造化し、補助金・助成金情報と継続的にマッチングし、年間目標金額を戦略的に獲得する。

対象：
- 親会社
- 100%子会社 2社
- 各社の複数事業部
- 経営管理部門
- 事業部門
- 必要に応じた外部専門家

設計思想：
- 管理部門が主導する
- 事業部門から案件情報を吸い上げる
- AIで探索・整理・ドラフト作成を支援する
- 最終判断は人間が行う
- 採択後の実績報告・入金までを成果として管理する
```

# 5. 目標金額から逆算する管理設計

## 5.1 年間目標の分解

例として、年間3,000万円の獲得を狙う場合。

| 区分          |                      金額 |
| ----------- | ----------------------: |
| 年間獲得目標      |             30,000,000円 |
| 必要申請総額      | 45,000,000円〜60,000,000円 |
| 想定採択・支給率    |                 50%〜70% |
| 四半期ごとの目標獲得額 |              7,500,000円 |
| 月次期待獲得額     |              2,500,000円 |

```markdown
| 区分 | 金額 |
|---|---:|
| 年間獲得目標 | 30,000,000円 |
| 必要申請総額 | 45,000,000円〜60,000,000円 |
| 想定採択・支給率 | 50%〜70% |
| 四半期ごとの目標獲得額 | 7,500,000円 |
| 月次期待獲得額 | 2,500,000円 |
```

## 5.2 管理すべき金額

```text
申請予定額：
申請候補として積み上げている金額

申請済額：
実際に提出した申請額

採択済額：
採択・支給決定された金額

交付決定額：
正式に交付決定された金額

実績報告後確定額：
実績報告後に確定した金額

入金済額：
実際に入金された金額
```

## 5.3 重要な考え方

```text
補助金・助成金獲得は、
「採択額」ではなく「入金済額」と「事業成果」で見る。

管理上は、
申請予定額 ＞ 申請済額 ＞ 採択済額 ＞ 交付決定額 ＞ 確定額 ＞ 入金済額
という減衰を前提にする。
```

# 7. 会議体設計

## 7.1 週次パイプライン会議

### 目的

案件を止めずに流す。

```text
頻度：
毎週1回

時間：
30〜45分

参加者：
- 経営管理部門
- 必要に応じて事業部責任者
- 必要に応じて外部専門家

議題：
1. 新着制度の確認
2. 締切30日以内の案件確認
3. 事業部への確認事項
4. 申請準備中案件の進捗確認
5. 採択後案件の証憑・実績管理
6. リスク・判断保留事項の確認
```

## 7.2 月次戦略会議

### 目的

年間目標に対する進捗を確認し、申請ポートフォリオを見直す。

```text
頻度：
月1回

時間：
60分

参加者：
- 経営層
- 経営管理部門
- 主要事業部責任者
- 必要に応じて外部専門家

議題：
1. 年間目標金額に対する進捗
2. 会社別・事業別の獲得状況
3. 申請予定額・申請済額・採択済額・入金済額
4. 次月以降の重点案件
5. 事業部別の案件創出状況
6. 不採択・失注理由の確認
7. 経営判断が必要な案件
```

```text
各事業部の施策に対して、活用可能な補助金・助成金を経営管理部門で探索・判定します。

事業部にお願いしたいのは、補助金を調べることではありません。

今後実施したい事業、投資、採用、研修、業務改善、DX、AI活用、省人化、販路開拓などの計画を、所定フォームに入力してください。

入力された内容をもとに、経営管理部門が補助金・助成金との適合性を確認し、申請可能性がある案件について個別に確認します。
```

# 15. リスク管理

| リスク     | 内容                  | 対策                |

# 23. 計画の要点

```text
この計画の本質：

補助金・助成金を「探す業務」ではなく、
自社の事業計画・投資計画・採用計画・研修計画を
公的支援制度に接続する経営管理プロセスとして構築する。

そのために必要なのは、
1. 自社情報の構造化
2. 事業案件の構造化
3. 制度情報の構造化
4. マッチング
5. 優先順位付け
6. 申請実行
7. 採択後管理
8. ナレッジ化

この8つを週次・月次で回すことで、
目標金額から逆算した補助金・助成金獲得パイプラインになる。
```
