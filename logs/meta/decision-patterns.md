```bash
# OpenAI モードで実行（デフォルト）
OPENAI_API_KEY=sk-... python scripts/meta_analysis.py

# Ollama モードで実行
META_MODE=ollama python scripts/meta_analysis.py
```

GitHub Actions の `.github/workflows/meta-analysis.yml` によって毎週木曜に自動実行されます。
