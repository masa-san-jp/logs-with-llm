# logs-with-llm Wiki

LLMとの会話ログを蓄積・活用するためのリポジトリです。

## このリポジトリについて

複数のLLM（Claude、Gemini、Grok など）との会話議事録を記録し、以下の目的に活用しています。

- 「誰と何を話したか」を忘れないための記録
- 過去の議事録をプロンプトとして再利用し、別スレッドや別LLMで会話を継続する
- 週次ブログの自動生成
- ドキュメント化ゴールの自動提案

## リポジトリ構成

| ディレクトリ/ファイル | 内容 |
|---|---|
| `logs/` | LLMとの会話議事録 |
| `blog/` | 自動生成された週次ブログ記事 |
| `prompts/` | 再利用するプロンプトテンプレート |
| `scripts/` | 自動化スクリプト |
| `.github/workflows/` | GitHub Actions ワークフロー |

## 基本的な使い方

1. LLMと会話する
2. 会話の最後に議事録作成プロンプトを送る
3. 出力された議事録を `logs/YYYYMMDD-{llm名}-{テーマ}.md` として保存
4. 次回は議事録をプロンプトに貼り付けて「この話の続きなんだけど…」と続ける
